## Installation

Clone this project:

```shell
git clone https://github.com/aasulaim/EEC136AB-Winter2022-Golf-Driver-Attention-Monitoring-System.git
```

### For Pip install
Install these dependencies (NumPy, OpenCV, Dlib):

```shell
pip install -r requirements.txt
```
### Verify Installation

Run the demo:

```shell
python GPIO_Demo.py
```
